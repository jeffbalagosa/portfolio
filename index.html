Oliver The Bot (A ChatGPT Clone)

I developed a clone of ChatGPT using React, Express, Cors, and Body Parser. The popularity of the original ChatGPT often results in server downtime. To overcome this, I leveraged GPT-3's API to build a similar website. I added an extra feature allowing users to choose their preferred model based on factors like cost and coding ability. By accessing the API as a paying customer, we're able to eliminate most downtime issues.

Since the app requires an API, I won't be putting this up as a live site. But you can check out the code on Github. It's a public repo and any keys within have been disabled or removed.

Arithmetic Generator

This app came out of wanting to build something to help support my daughter's new love for math. It came out better than I hoped, so I wanted to showcase it here. The app generates random addition and subtraction problems.

I gamiefied it by placing a streak counter on the app. The more in a row you got correct, the higher the number grows. It goes back to zero when you submit a wrong answer. She enjoys playing it!
